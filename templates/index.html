Control de Tráfico Inteligente

Bienvenido a nuestra plataforma de gestión de tráfico inteligente. Este sistema combina tecnologías modernas como ESP32 y sensores ultrasónicos para optimizar la gestión del tráfico en tiempo real.

El sistema funciona detectando vehículos frente a los semáforos y utilizando un autómata programado para controlar el flujo vehicular de manera eficiente. Los datos capturados se envían a un servidor remoto que almacena la información en una base de datos.

A través de esta aplicación web, los usuarios pueden:

  • Visitantes: Consultar información de tráfico en tiempo real mediante gráficos interactivos.
  • Administradores: Gestionar remotamente los semáforos, analizar estadísticas detalladas del tráfico y monitorear la actividad de los visitantes.

Nuestro objetivo es proporcionar una herramienta intuitiva y poderosa para mejorar la movilidad urbana y la experiencia de los usuarios.
